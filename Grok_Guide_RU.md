# 🔥 Grok SDXL Prompt Batch (Официальное Руководство)

Ноды **Grok SDXL Prompt Batch** предназначены для профессиональной генерации пакетов промптов для Stable Diffusion XL. Они используют возможности ИИ Grok от x.ai для создания семантически богатых, синтаксически правильных (SDXL-формат) и контекстно-зависимых описаний.

В проекте доступны **две версии** генератора промптов, созданные для разных задач:

### 1. 🌈 Grok SDXL Prompt Batch (Diverse)
*   **Задача**: Создание **максимально разнообразных** людей и сцен.
*   **Поведение ИИ**: Гроку даны жесткие инструкции *принудительно* и случайным образом менять цвет волос, этническую принадлежность, возраст (от 18 до 60) и тип телосложения для каждого генерируемого промпта. Никаких клонов.

### 2. 👯‍♀️ Grok SDXL Prompt Batch (Identical)
*   **Задача**: Создание **максимально одинаковых** персонажей для раскадровок или комиксов.
*   **Поведение ИИ**: Гроку даны жесткие инструкции *сохранять* все физические характеристики (лицо, одежда, раса) полностью идентичными во всем батче.

---

## 🛠️ Входные параметры (Inputs)

### 🔴 Обязательные (Required)

1.  **user_task (STRING)**
    *   **Суть**: Ваша инструкция для ИИ на естественном языке.
    *   **Пример**: `"Напиши 20 промптов: девушка киберпанк, неоновое освещение"`.

2.  **count (INT)**
    *   **Суть**: Общее количество промптов, которое вы хотите получить на выходе.
    *   **Синхронизация**: Число автоматически передается в Batch интерфейс ComfyUI (только в связке с нашими лоадерами).

3.  **model (STRING)**
    *   **Значение по умолчанию**: `grok-4-1-fast-non-reasoning`.

4.  **temperature (FLOAT)**
    *   **Суть**: Степень "случайности" или креативности ответов. (0.7-1.0 — оптимально).

---

## 🚀 Работа с LoRA (Intelligent Selection)

### **lora_mode**
*   **off**: Функционал LoRA полностью отключен.
*   **auto**: ИИ получает каталог ваших LoRA и сам решает, какие из них подходят под задачу. Грок вставит триггеры именно там, где они уместны.
*   **force**: Заставляет ИИ в обязательном порядке использовать хотя бы одну LoRA в каждом промпте.

---

## 📤 Выходы (Outputs)

1.  **prompts_text (STRING)**: Все промпты одной большой строкой.
2.  **used_loras (STRING)**: JSON-список выбранных LoRA (для подключения к Variant A).
3.  **loras_with_ai_strength (STRING)**: JSON-список с силой от ИИ (для подключения к Variant B).
4.  **count_generated (INT)**: Количество созданных промптов (подключайте к Empty Latent Image).

---

## 🔗 Пакет Нод: Grok Lora Loaders

Для работы ИИ с вашими моделями используйте один из двух лоадеров:

### 1. 🤖 Grok Lora Loader (Auto-Text)
**Вариант А: Грок выбирает LoRA, вы задаете силу.**

*   **Как подключить**: Соедините выход **`used_loras`** со входом лоадера.
*   **Вход `default_strength`**: Сила по умолчанию (обычно 1.0).
*   **Вход `lora_strengths`**: Текстовое поле для ручной корректировки.
    *   *Пример:* `cyberpunk_v1: 1.2`

### 2. 🧠 Grok Lora Loader (AI Strategy)
**Вариант Б: Грок сам подбирает силу LoRA.**

*   **Как подключить**: Соедините выход **`loras_with_ai_strength`** со входом лоадера.
*   **Вход `strength_override`**: Поле для вмешательства в логику ИИ.
    *   *Пример:* `princess_xl_v2: 0.9`

## 🛡️ Нода: Grok Image Saver

Эта нода работает в точности как стандартный Save Image в ComfyUI, сохраняя ваши финальные генерации на диск. У неё есть две супер-функции:

1.  **Абсолютная безопасность (Без метаданных)**: Принудительно удаляет весь "workflow" (схему нод, настройки, скрытые теги Грока и метаданные) из файла PNG при сохранении. Если вы выкладываете свои работы в сеть, никто не сможет украсть ваш промпт или структуру.
2.  **Водяной знак**: Вы можете вписать в поле `watermark_text` любую строку (свой сайт, телеграм-канал, имя). Нода автоматически, красиво и ненавязчиво наложит этот текст в правый нижний угол изображения. Текст динамически масштабируется под любое разрешение и имеет умную полутень, поэтому отлично читается как на светлом, так и на темном фоне.

## 📏 Нода: Grok SDXL Aspect Ratio

Это супер-удобная нода-помощник для задания размеров генерации без необходимости вспоминать или вручную вводить пиксели. Она подключается напрямую ко входу `Width` и `Height` стандартной ноды `Empty Latent Image`.

*   **Как использовать**: Просто выберите нужный формат из выпадающего списка. Все форматы представлены с интуитивными пиктограммами и точными SDXL-оптимизированными разрешениями:
    *   `🔳 1:1 (1024 x 1024)` — Квадрат (Идеально для аватарок)
    *   `🖥️ 16:9 (1344 x 768)` — Широкий формат (Youtube, Обои)
    *   `📱 9:16 (768 x 1344)` — Портретный (TikTok, Reels, Shorts)
    *   `🖼️ 3:2 (1216 x 832)` — Пейзаж (Классическое фото)
    *   `📝 2:3 (832 x 1216)` — Вертикальное фото
    *   `🎦 21:9 (1536 x 640)` — Ультра-широкий (Кинематографичный)
    *   `📸 4:3 (1152 x 896)` — Стандартная камера

Эта нода полностью избавляет от ошибок с пропорциями при работе с SDXL моделями.

## ️ Нода: Grok Batch Image Gallery

Специальный просмотрщик для пакетов (batch) картинок.

*   **Увеличение по клику**: Просто кликните левой кнопкой мыши по любой картинке в сетке, и она увеличится на всё окно ноды!
*   **Удобство**: Показывает результаты всей пачки (например, 4 или 8 штук) в виде красивой сетки.
